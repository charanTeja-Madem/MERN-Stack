Week 1: JavaScript Fundamentals
This week covers the core concepts of JavaScript, including basic data structures and control flow.

Topics Covered
1. Array Operations
Simple Arrays (Array_simple/):
temperature_analytics.js: Basic filtering, mapping, and reducing on thermal data.
marks_basic_ops.js: Array methods applied to student marks.
Advanced Arrays (Array_adv/):
cart_operations.js: Managing e-commerce shopping carts.
student_grades.js: Calculating grades for a list of students.
employee_salary.js: Processing payroll and department filtering.
movie_filtering.js: Analyzing movie metadata and ratings.
transaction_processing.js: Credit/Debit balance calculations.
2. Control Statements (Control_Statements/)
login_status.js: Conditional checks for user authentication.
pricing_categories.js: Tiered pricing logic (Budget, Standard, Premium).
enrollment_checker.js: Ternary operator usage for course enrollment.
3. Object Operations (Object_op/)
user_profile_management.js: CRUD operations on user objects.
marks_analytics.js: Calculating statistics from object-based scores.
4. Operators (Operators/)
billing_calculator.js: Compound operations for GST and bill calculation.
