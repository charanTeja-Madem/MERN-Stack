Week 3: Backend Development with Node.js & Express
This week introduces backend development concepts, focusing on building RESTful APIs using Express and integrating with a MongoDB database.

Projects
1. BACKEND
A comprehensive backend setup featuring:

Express Server: Configured in server.js.
Mongoose Integration: Connecting to MongoDB and defining schemas/models in the models/ directory.
Modular APIs: Route handlers organized in the APIs/ directory (e.g., userapi.js, productApi.js).
REST Testing: Includes res.http for testing API endpoints using REST Client.
2. BACKEND-DEMO-1
A simplified demo project for understanding the basic structure of an Express application and API routing.

Key Features
Database connection using Mongoose.
CRUD operations for Users and Products.
Modular routing for cleaner code organization.
Error handling in async database operations.
Setup Instructions
Navigate to the project directory:
cd week-3/BACKEND
Install dependencies:
npm install
Start the server:
node server.js
